\# 🌍 Job Market Analytics Dashboard



An end-to-end Data Analytics project built using \*\*SQL Server, Power BI, DAX, and Python\*\* to analyze global workforce trends, candidate profiles, salary insights, and hiring performance.



\---



\## 📌 Project Overview



This project provides business insights into:



\- Candidate demographics and education

\- AI specialization trends

\- Salary expectations and offers

\- Hiring outcomes and recruitment performance



The dashboard helps stakeholders understand workforce distribution and hiring efficiency through interactive visualizations.

\## 📊 Dashboard Pages



\### 1️⃣ Executive Dashboard

Provides a high-level overview of:



\- Total Candidates

\- Average Salary

\- Average Technical Score

\- Average Experience



\---



\### 2️⃣ Candidate Analytics

Analyzes:



\- Gender distribution

\- University tiers

\- Major distribution

\- Experience levels

\- Kaggle performance



\---



\### 3️⃣ Salary Analytics

Provides insights into:



\- Salary trends by industry

\- Company ratings

\- Expected vs Offered salary comparison

\- Offer acceptance rate



\---



\### 4️⃣ Hiring Performance

Evaluates:



\- Technical scores

\- Interview scores

\- Coding test scores

\- Hiring rounds

\- Hiring success rate



\---



\## 🔍 Key Insights



\- Candidates with higher technical scores have better hiring outcomes.

\- Interview scores strongly influence final hiring decisions.

\- Salary expectations vary across industries.

\- AI specialization impacts candidate distribution.

\- Company ratings and salary offers affect acceptance rates.
